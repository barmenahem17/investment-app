# Investment App (v1)

אפליקציה אישית לניהול 3 תיקים (Extrade, IBKR, Kraken), פירוק ביצועים: רווחי שוק, רווח/הפסד מהמרות דולר/שקל, והפקדות; אפשרות לבחור מטבע ראשי (ILS/USD).

## Stack
- Next.js (App Router, TypeScript)
- Tailwind CSS + shadcn/ui
- Prisma + SQLite
- Vitest

## Setup
```bash
npm install
npm run dev
```
